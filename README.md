## 프로젝트 개요

Todo-List

- signup / login (LocalStorage)
- todo list CRUD

## 프로젝트 실행 방법

이 프로젝트는 Create React App + TypeScript 로 만들어 졌습니다.

```shell
npm install
npm run start
```

다음 명령어를 입력 시 development mode 가 실행됩니다.

http://localhost:3000 에서 확인하실 수 있습니다.
